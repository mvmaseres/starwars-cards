PURPOSE

DISCOVERING GALACTIC JOY AND WISDOM

Born from a passion for Star Wars, our card game aims to engage and entertain enthusiasts. With diverse guessing options and difficulty levels, we offer an immersive experience, making a meaningful contribution to the Star Wars community.

API

EXPLORATION WITH SWAPI

Our primary data source for the game is the Star Wars API, affectionately known as "swapi." This API is the first quantified and programmatically accessible data source for all information in the canonical Star Wars universe. By leveraging this API, we can offer players a diverse range of options for guessing and exploring in our card game. For more details on how to interact with the API and the data available, visit swapi.dev.
